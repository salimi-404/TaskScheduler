# 🚀 شبیه‌ساز زمان‌بند وظایف (Task Scheduler Simulator)

این پروژه یک سیستم شبیه‌سازی هوشمند برای مدیریت و زمان‌بندی وظایف (Tasks) است که با رعایت اصول وابستگی‌ها (Dependencies)، اولویت‌بندی (Priority) و بررسی مهلت زمانی (Deadline) طراحی شده است.

---

## 📂 ساختار پروژه
- `models/`: شامل کلاس‌های پایه (مانند `Task`).
- `scheduler/`: هسته اصلی منطق زمان‌بندی و تشخیص چرخه‌ها (Cycles).
- `tests/`: تست‌های واحد برای تضمین صحت عملکرد سیستم.
- `main.py`: نقطه ورود و اجرای برنامه.

## ✨ ویژگی‌های کلیدی
*   ✅ **مدیریت اولویت‌ها:** صف‌بندی هوشمند وظایف بر اساس اولویت عددی.
*   ✅ **تشخیص وابستگی:** استفاده از الگوریتم‌های گراف برای جلوگیری از چرخه‌های بی‌پایان.
*   ✅ **مدیریت مهلت زمانی:** حذف خودکار وظایف منقضی شده و مسدود کردن وابستگان آن‌ها.
*   ✅ **تست‌محور:** دارای تست‌های واحد جامع برای اطمینان از صحت منطق برنامه.

## 🛠 نحوه اجرا
برای اجرای برنامه در ترمینال، از دستور زیر استفاده کنید:
```bash
python main.py
```

برای اجرای تست‌ها و اطمینان از سلامت کد:
```bash
python -m unittest discover tests
```
