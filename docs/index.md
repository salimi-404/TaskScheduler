# شبیه‌ساز زمان‌بندی وظایف (Task Scheduler) 🚀

به مستندات فنی و راهنمای کاربری پروژه خوش آمدید. این سیستم یک ابزار پیشرفته و ماژولار است که برای شبیه‌سازی، تحلیل و مدیریت اولویت‌بندی وظایف پیچیده با استفاده از زبان **Python** طراحی شده است.

---

## ✨ قابلیت‌های کلیدی سیستم

!!! success "مدیریت وابستگی‌ها (Task Dependencies)"
    کارها بر اساس گراف وابستگی اجرا می‌شوند؛ یعنی اگر وظیفه B به وظیفه A وابسته باشد، سیستم به صورت هوشمند تا اتمام کار A منتظر می‌ماند.

!!! info "اولویت‌بندی و مدیریت مالی"
    سیستم قادر است بر اساس سه فاکتور کلیدی: **اولویت (Priority)**، **ددلاین (Deadline)** و **سود مالی کار (Profit)** بهترین چیدمان زمانی را شبیه‌سازی کند.
