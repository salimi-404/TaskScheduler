# 🚀 شبیه‌ساز هوشمند زمان‌بندی وظایف | Task Scheduler Simulator

یک سیستم پیشرفته، ماژولار و شیءگرا به زبان پایتون برای شبیه‌سازی, تحلیل و مدیریت اولویت‌بندی وظایف پیچیده بر اساس گراف وابستگی‌ها (DAG)، مهلت‌های زمانی (Deadlines) و بهینه‌سازی سود مالی (Profit).

---

## ✨ ویژگی‌های کلیدی سیستم

* **🔗 مدیریت گراف وابستگی‌ها:** بررسی خودکار روابط بین تسک‌ها و جلوگیری از بروز بن‌بست (Deadlock) یا چرخه‌های بی‌انتها با الگوریتم‌های گراف.
* **⏳ زمان‌بندی مبتنی بر Slack Time:** صف‌بندی هوشمند و پویا در صف اولویت‌دار (Min-Heap) بر اساس ددلاین و زمان باقی‌مانده.
* **💰 بهینه‌سازی سود مالی (Profit):** چیدمان کارها به طوری که بیشترین بازدهی مالی در محدودیت‌های زمانی فشرده حاصل شود.
* **💥 مدیریت آبشاری تسک‌های سوخته:** مسدود کردن خودکار و زنجیره‌ای تمام تسک‌های وابسته، در صورتی که ددلاین تسک مادر بسوزد.
* **🧪 پایدار و تست‌محور:** پوشش کامل منطق برنامه با تست‌های واحد (Unit Tests) جامع.

---

## 📂 ساختار ماژولار پروژه

```text
TaskSchedulerSimulator/
├── docs/               # فایل‌های منبع مستندات (Markdown)
├── models/             # کلاس‌های پایه و ساختار داده‌ای تسک‌ها (Task)
├── scheduler/          # هسته مرکزی منطق زمان‌بندی و مدیریت صف اولویت
├── tests/              # تست‌های واحد برای اعتبارسنجی فرآیندها
├── main.py             # نقطه ورود و اجرای سناریوهای شبیه‌سازی
├── mkdocs.yml          # کانفیگ و تنظیمات قالب مستندات
└── README.md           # راهنمای فعلی پروژه
```

## 🛠 راهنمای راه‌اندازی و اجرا

### ۱. اجرای شبیه‌ساز
برای اجرای سناریوی پیش‌فرض و مشاهده نحوه زمان‌بندی تسک‌ها در ترمینال، دستور زیر را وارد کنید:
```bash
python main.py
```

### ۲. اجرای تست‌های واحد
برای اطمینان از سلامت کامل منطق برنامه و بخش‌های مختلف زمان‌بند:
```bash
python -m unittest discover tests
```
